HTML

HTML(Hypertext Markup Language) is a markup language used for creating the structure and content of web page. It is the standard language for building websites and is supported by all web browsers. HTML uses a set of tags to define the structure and layout of a web page. These tags are enclosed in angle brackets (<>) and are interpreted by web browsers to display the content in a structured way. HTML tags can be used to define headings, paragraphs, lists, images, links, tables, forms, and more.

CSS

CSS(Cascading Style Sheet) is a stylesheet language used for describing the presentation and styling of a document written in HTML. It is used to control the layou, appearance, and visual aspects of web pages.CSS works by applying styles to HTML elements, allowing web developers to coontrol the font, color, size, spacing, positioning, and other visual properties of elements on a web page. Here are some features and concepts of CSS: 1. Selectors: CSS uses selectors to target specific HTML elements. Selectors can be based on element types, classes, IDs, attributes, and more.

Parts of URL

A URL(Uniform Resource Locatore) is a string of characters that provides the address and location of a resource on the internet. It consists of several parts, each serving a specific purpose. Parts of URL: 1. Scheme- It tells web servers which protocol to use when it accesses a page on your websites. 2.Domain name- It is the unique reference that identifies a website on the internet. It always include a subdomain, second-level domain, and top-level domain. a). Subdomain- It indicates which particular page of your website the web browser should serve up. The used of subdomain above is www but there are pther subdomain that can be used. b). Second-level domain- It is the name of the websites. c).Top-level domain- It specifies what type of entity your organization register as on the internet. Endings of web pages tells us a bit about the page. 3.Port- The port number is rarely visible in URLs but always required. When declared is a URL, it comes right after top-level domain, seperated by a colon. 4.File Path/Path/Directory- The path typically refers to a file or directory on the web server. It directs the browser to a specific page within the site. 5.Parameter String/Parameter- Some URLs include a string of characters after the path - beginning with a question mark. 6.Anchor: The fragment or anchor refers to a specific section or location within a web page.
